四、环境踩坑记录

4.1 DNS 修复

服务器使用 Tailscale DNS（100.100.100.100），无法解析外网域名。

修复：追加阿里云公共 DNS

4.2 pip 安装失败

Python 3.6.8 + 外网受限，pip 和 dnf 均无法正常使用。Tsighua/Aliyun 镜像也超时。

解决：放弃所有 pip 依赖，用 Python 3.6 标准库重写整个后端。

4.3 Nginx SSL 模块缺失

宝塔编译的 Nginx 未包含 ngx_http_ssl_module，但原配置中有 listen 443 ssl 指令。这导致 nginx -t 失败，无法 reload。

解决：移除整个 HTTPS server 块。ir.hwarkm.com 只用 HTTP 80 端口，HTTPS 由 CDN/反向代理层处理（如有需要）。

4.4 SSH 密钥配置

Windows 本地 SSH 密钥未在服务器 authorized_keys 中，无法免密登录。

修复（在宝塔终端执行）：

4.5 Python 3.6 兼容性

  • openai 包依赖 pandas>=1.2.3，后者需要 Python ≥ 3.7
  • fastapi 需要 Python ≥ 3.7
  • scikit-learn 1.3+ 需要 Python ≥ 3.8
  • 系统中同时有 Python 3.7（pyenv）和 Python 3.11（/usr/bin），但均为残损安装

最终方案：Python 3.6 标准库（http.server + urllib.request + json）零依赖实现全部功能。

七、演示脚本

比赛现场演示流程：

  1. 打开页面 → http://ir.hwarkm.com

    • 左栏：视频监控区域（Pi 推流后可显示实时画面 + 检测框）
    • 中栏：6 步 S摩托车发动机维修流程，步骤 1 已完成(绿)，步骤 2 进行中(蓝)
    • 右栏：AI 助手显示”● RAG 已连接”

  2. 展示文档检索 → 输入”缸盖螺栓的标准力矩是多少？”

    • 系统检索到手册第 9.1 节
    • DeepSeek 回答：18-22 N·m，并补充注意事项

  3. 展示步骤联动 → 点击步骤 3”零部件检查与测量”

    • 状态变为 active（蓝底高亮）
    • 展开详情：显示气缸内径测量方法、椭圆度标准值
    • 步骤 2 自动变为 completed（绿色）

  4. 展示多轮对话 → 追问”如果间隙超标怎么办？”

    • 系统结合前文上下文 + 手册章节回答：”更换活塞环，安装时环口错开 120°”

  5. 展示告警 → 等待或手动触发

    • 弹窗：安全风险告警
    • 点击”查看相关步骤”跳转到对应 SOP 步骤
